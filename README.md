# Jornada - Busca de Passagens

Projeto desenvolvido com Angular 19 e Angular Material, baseado em um layout do Figma, com foco na construção de interfaces e componentização.

## Tecnologias

* Angular 19
* Angular Material
* TypeScript
* SCSS
* HTML

## Objetivo

Praticar o desenvolvimento de interfaces utilizando Angular, com ênfase em organização de componentes, uso de Angular Material e boas práticas de front-end.

## Funcionalidades

* Interface de busca de passagens
* Campos de origem e destino
* Seleção entre ida e volta ou somente ida
* Estrutura de navegação com header


## Observações

O design da interface foi fornecido por um layout no Figma e não é de autoria própria.

## Créditos

Projeto desenvolvido com base em curso da Alura.
